# 🏠 住宅ローン・人生設計ツール

**Mortgage & Life Planning Simulator** — 住宅ローン選定から死ぬまでの資産形成を一気通貫でシミュレーションするパーソナルファイナンスツール。

![Python](https://img.shields.io/badge/Python-3.10%2B-blue) ![Streamlit](https://img.shields.io/badge/Streamlit-1.32%2B-red) ![License](https://img.shields.io/badge/License-MIT-green)

---

## 📋 機能概要

| モジュール | 機能 |
|---|---|
| 🏦 **銀行金利取得** | AIプロンプト自動生成 → JSON貼り付けで最新金利を取得・検証 |
| 📊 **ローン比較** | 9行横断比較テーブル・ブレークイーブン分析・金利上昇シナリオグラフ |
| ⚡ **繰り上げ返済** | 期間短縮/月額減額型・複数回設定・繰り上げvs投資比較・借り換え計算 |
| 🗺️ **人生設計** | 生涯キャッシュフロー・ライフイベント・楽観/ベース/悲観3シナリオ・CSV出力 |

## 🚀 セットアップ

### 1. リポジトリをクローン

```bash
git clone https://github.com/mineon09/fp_plan.git
cd fp_plan
```

### 2. 依存パッケージをインストール

```bash
pip install -r requirements.txt
```

> **必要パッケージ:** streamlit / pandas / numpy / plotly / scipy / fpdf2

### 3. アプリを起動

```bash
streamlit run app.py
```

ブラウザで `http://localhost:8501` が自動で開きます。

## 📖 使い方

### Step 1 — 🏦 銀行金利を取得する

1. サイドバーの **「🏦 銀行金利取得」** を選択
2. 対象銀行を選択し **「プロンプトを生成」** をクリック
3. 生成されたプロンプトを Claude / ChatGPT に貼り付けて実行
4. 返ってきたJSONをテキストエリアに貼り付けて **「パース・検証」**
5. テーブルを確認して **「このデータで計算する」** をクリック

> デフォルト値（2024年末参考値）を使う場合は **「デフォルト値を使用」** をクリック

### Step 2 — 📊 ローン条件を比較する

- サイドバーで借入金額・期間・ボーナス返済割合を設定
- 全銀行の月額返済・総利息・実効金利を比較テーブルで確認
- **ブレークイーブン分析**: 変動金利が何%を超えると固定が有利かを自動計算
- **シナリオグラフ**: 金利上昇時（+1〜+3%）の累計返済額をPlotlyで可視化

### Step 3 — ⚡ 繰り上げ返済・借り換えをシミュレート

| タブ | 内容 |
|---|---|
| 繰り上げ返済 | 任意の月・金額で複数回設定、節約利息・短縮期間を計算 |
| 繰り上げ vs 投資 | 同じ資金をNISA/インデックス投資に回した場合との比較 |
| 借り換え | 残債・現行金利・借り換え先金利から損益分岐月数を計算 |

### Step 4 — 🗺️ 人生設計を立てる

1. 基本プロファイル（年収・定年・年金・現在資産など）を入力
2. ライフイベント（教育費・車・リフォームなど）をプリセットから選択・編集
3. **「シミュレーション実行」** → 90歳までの純資産推移グラフを表示
4. サマリーカードで完済年齢・資産ピーク・老後枯渇判定・FIRE達成年齢を確認
5. 年次収支テーブルを **CSV ダウンロード** で保存

## 🧮 主要計算仕様

| 計算 | 仕様 |
|---|---|
| 月額返済（元利均等） | `M = P × r(1+r)^n / ((1+r)^n - 1)` |
| 金利上昇シナリオ | 残高ベースで毎年再計算（線形上昇） |
| 実効金利（団信込み） | 適用金利 + 0.25%（団信保険料相当） |
| ブレークイーブン金利 | SciPy brentq 数値解 |
| 生涯CF | 年次ステップ・インフレ/年収成長率を複利適用 |

### シナリオ設定

| シナリオ | 金利上昇 | 投資利回り | インフレ | 年収成長 |
|---|---|---|---|---|
| 楽観 | +0.5% | 6.0% | 1.5% | +2.0% |
| ベース | +1.0% | 4.0% | 2.0% | +1.5% |
| 悲観 | +2.5% | 2.0% | 3.0% | +0.5% |
